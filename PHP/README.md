# N-Queens problem using backtracking and CS

## PHP implementation

### Set up (Ubuntu 22.04)

```shell
apt install php
```

### Syntax

```
queens.php [-test] [SIZE]
```

<dl>
  <dt>-test</dt>
  <dd>Enable test output.</dd>
  <dt>SIZE</dt>
  <dd>Length of the chess.</dd>
</dl>

### Example

```
# php queens.php 256

Queen 1: square 172
Queen 2: square 1003
(...)
Queen 255: square 45
Queen 256: square 174
Solved in 5106 steps. Time: 445.57 ms.
```
