# N-Queens problem using backtracking and CS

## Python implementation

### Set up (Ubuntu 22.04)

```shell
apt install python3
```

### Syntax

```
queens.py [-test] [SIZE]
```

<dl>
  <dt>-test</dt>
  <dd>Enable test output.</dd>
  <dt>SIZE</dt>
  <dd>Length of the chess.</dd>
</dl>

### Example

```
# ./queens.py 512

Queen 1: square 172
Queen 2: square 1003
(...)
Queen 511: square 1
Queen 512: square 1
Solved in 1591 steps. Time: 1241.388 ms.
```
